Problem 2: Shuffle String (LeetCode #1528)
Difficulty: Easy
Link: https://leetcode.com/problems/shuffle-string/
Problem Statement
Given string s and array indices, place each character at position indices[i].
Example:
Input: s = "codeleet", indices = [4,5,6,7,0,2,1,3]
Output: "leetcode"
Solution
javaclass Solution {
    public String restoreString(String s, int[] indices) {
        char[] result = new char[s.length()];
        
        for (int i = 0; i < s.length(); i++) {
            result[indices[i]] = s.charAt(i);
        }
        
        return new String(result);
    }
}
Explanation

Create char array for result
Place each character from position i to position indices[i]
Convert char array to String

Time Complexity: O(n)
Space Complexity: O(n)
