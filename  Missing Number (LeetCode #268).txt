Problem 4: Missing Number (LeetCode #268)
Difficulty: Easy
Link: https://leetcode.com/problems/missing-number/
Problem Statement
Given array containing n distinct numbers in range [0, n], return the missing number.
Example:
Input: nums = [3,0,1]
Output: 2
Explanation: Range is [0,3], missing number is 2
Solution 1: Using Sum Formula (Best)
javaclass Solution {
    public int missingNumber(int[] nums) {
        int n = nums.length;
        int expectedSum = n * (n + 1) / 2;
        int actualSum = 0;
        
        for (int num : nums) {
            actualSum += num;
        }
        
        return expectedSum - actualSum;
    }
}
Explanation:

Sum of 0 to n = n Ã— (n + 1) / 2
Calculate actual sum of array
Missing number = Expected - Actual

Time Complexity: O(n)
Space Complexity: O(1)
Solution 2: Using XOR
javaclass Solution {
    public int missingNumber(int[] nums) {
        int xor = nums.length;
        
        for (int i = 0; i < nums.length; i++) {
            xor ^= i ^ nums[i];
        }
        
        return xor;
    }
}
Explanation:

XOR property: a ^ a = 0, a ^ 0 = a
XOR all indices and values
Duplicate pairs cancel out, missing remains

Time Complexity: O(n)
Space Complexity: O(1)
