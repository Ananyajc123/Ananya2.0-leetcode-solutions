## Problem 4: Move Zeroes (LeetCode #283)

**Difficulty:** Easy  
**Link:** https://leetcode.com/problems/move-zeroes/

### Problem Statement
Given an integer array `nums`, move all `0`'s to the end while maintaining the relative order of non-zero elements.

**Example:**
```
Input: nums = [0,1,0,3,12]
Output: [1,3,12,0,0]
Solution
javaclass Solution {
    public void moveZeroes(int[] nums) {
        int nonZero = 0;  // Position for next non-zero
        
        // Move all non-zeros to front
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != 0) {
                nums[nonZero] = nums[i];
                nonZero++;
            }
        }
        
        // Fill rest with zeros
        for (int i = nonZero; i < nums.length; i++) {
            nums[i] = 0;
        }
    }
}
Explanation:

First pass: collect all non-zero elements at front
Second pass: fill remaining positions with zeros
Maintains relative order

Alternative (Swap approach):
javaclass Solution {
    public void moveZeroes(int[] nums) {
        int nonZero = 0;
        
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != 0) {
                // Swap
                int temp = nums[nonZero];
                nums[nonZero] = nums[i];
                nums[i] = temp;
                nonZero++;
            }
        }
    }
}
```

**Time:** O(n)  
**Space:** O(1)
