# Valid Palindrome

Problem from LeetCode.

## Problem Statement

A phrase is a palindrome if, after:

* converting all uppercase letters to lowercase
* removing all non-alphanumeric characters

it reads the same forward and backward.

Alphanumeric characters include:

* letters (`a-z`, `A-Z`)
* numbers (`0-9`)

Return `true` if the string is a palindrome, otherwise return `false`.

# Approach — Two Pointer Technique

We use:

* one pointer from the beginning
* one pointer from the end

Steps:

1. Skip non-alphanumeric characters
2. Convert characters to lowercase
3. Compare both characters
4. Move pointers inward

If any mismatch occurs, return `false`.

---

# Java Solution

```java id="r8e1j5"
class Solution {
    public boolean isPalindrome(String s) {

        int left = 0;
        int right = s.length() - 1;

        while (left < right) {

            // Skip invalid characters from left
            while (left < right &&
                  !Character.isLetterOrDigit(s.charAt(left))) {
                left++;
            }

            // Skip invalid characters from right
            while (left < right &&
                  !Character.isLetterOrDigit(s.charAt(right))) {
                right--;
            }

            // Compare lowercase characters
            if (Character.toLowerCase(s.charAt(left)) !=
                Character.toLowerCase(s.charAt(right))) {

                return false;
            }

            // Move inward
            left++;
            right--;
        }

        return true;
    }
}
```
