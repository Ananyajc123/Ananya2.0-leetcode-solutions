Problem 2: Power of Two (LeetCode #231)
Difficulty: Easy
Link: https://leetcode.com/problems/power-of-two/
Problem Statement
Return true if n is a power of two.
Example:
Input: n = 16
Output: true
Explanation: 2^4 = 16
Solution (Recursive)
javaclass Solution {
    public boolean isPowerOfTwo(int n) {
        if (n <= 0) return false;
        if (n == 1) return true;
        if (n % 2 != 0) return false;
        
        return isPowerOfTwo(n / 2);
    }
}
